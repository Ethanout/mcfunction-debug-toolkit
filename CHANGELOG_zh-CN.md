# 更新日志

[English](CHANGELOG.md) | **简体中文**

MCFunction Debug Toolkit 的所有重要变更都会记录在本文件中。项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.1] - 2026-07-15

### 新增

- 新增 `{@s}`、`{@e[tag=target]}` 等原版选择器简写，无需额外查询关键字即可简洁输出实体名称。
- 支持纯文本 `#!` 行，可用于标签、分隔线、字面花括号和结构化多行调试输出。
- 原版风格的值颜色：数字为金色、字符串与未设置样式的实体名称为绿色、资源标识符为青色、调用栈分隔符为灰色，并使用原生 SNBT 高亮。
- 新增独立的 AI Agent 安装指南、发布流程、更新日志，以及两种 README 中的真实游戏截图。

### 变更

- 重写英文和简体中文 README，围绕简洁实用的数据包调试示例展开，并把 MCP 定位为可选能力。
- 项目版本统一由根目录 `package.json` 提供；Fabric 产物名、Fabric 元数据和 MCP 握手都从该值派生。
- 扩展忽略规则，覆盖凭据、本地环境、生成输出、覆盖率数据和操作系统元数据。

### 修复

- `.1f` 等数字格式会继续保留金色数值样式，不再丢失嵌套 SNBT 颜色。
- 选择器和分数输出会保留原生实体队伍颜色、hover、click 与 insertion 元数据。
- 选择器求值与其他集合查询使用相同的有界来源与叶子预算。

## [0.1.0] - 2026-07-15

以 MCFunction Debug Toolkit 名义发布的首个公开开发预览版。

### 新增

- 面向人的 `.mcfunction` `#!` 调试指令，可查询执行上下文、记分板、实体 NBT、方块 NBT 和命令 storage。
- 多行模板、转义字面花括号、重复 `...` pattern、嵌套列表分组、`/strip`、`/no_strip` 和紧凑数字格式。
- 结构化调试事件、reload 诊断、简短的游戏内运行时错误、函数名和嵌套函数调用栈。
- 英文与简体中文项目文档、协议说明、架构笔记、安全指南和真实数据包测试。

### 变更

- 将 MCP 重新定位为独立数据包调试模组周围的可选自动化层。
- 为渲染叶子、NBT 字节、组件大小、事件历史、聊天历史、截图、命令反馈和输入持续时间添加上限。

### 修复

- 错误指令在 reload 时会被报告并跳过，不影响函数中其余命令加载。
- 客户端关闭、失焦和请求取消都会干净释放按住的按键与鼠标按钮。

## [0.0.0] - 2026-07-14

用于验证项目架构的内部概念验证版本。

### 新增

- 面向 Minecraft Java 26.2 的初始 Fabric 模组与 Node.js MCP 服务器工作区。
- 用于服务端命令和客户端操作、经过身份验证的本机回环 HTTP 桥接。
- 实时 Brigadier 命令验证、单命令执行和有界顺序批处理。
- 增量 HUD 聊天捕获、JPEG 截图、键鼠输入、视角移动、单击、双击、长按和拖动。
- 共享 bearer token 发现、确定性危险命令护栏、超时报告、有界响应分页和初始自动化测试。
