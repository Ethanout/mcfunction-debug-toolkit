# 路线图

[English](ROADMAP.md) | **简体中文**

## 已在 0.2 完成

- 增量捕获 HUD 聊天，包括 `tellraw` 文本。
- 客户端按键、鼠标、视角与等待操作序列，并自动释放输入。
- 窗口失去焦点时释放鼠标。
- 通过 MCP 返回并保存 JPEG 游戏画面截图。

## 已在 0.3 完成

- 面向人的 `#!` 调试指令，可查询上下文、分数和 NBT。
- 多行模板、嵌套来源/值列表、重复与 strip 行为，以及有界数字格式化。
- reload 阶段使用原版语法验证，并映射到真实源码位置。
- 原生聊天组件、简短运行时错误、有界结构化事件，以及通过 MCP 暴露的 reload 诊断。
- 共享的叶子与字符截断预算，并提供可见截断标记。
- 当前函数和同步函数调用栈追踪，包括 schedule 创建的新栈根。
- 客户端 HTTP 生命周期可干净关闭。
- 单调有界的聊天/调试游标，并检测历史丢失。
- 整个输入请求预验证、总超时和失焦取消。
- 角度单位正确的视角控制与 ARGB 转换正确的 JPEG 截图。
- 服务端/客户端桥接共享令牌，并强化 HTTP 请求边界。
